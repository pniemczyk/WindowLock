How It Works

WindowLock sits in your menu bar and quietly keeps track of every window — which app, which monitor, what position and size. One click to save a layout. One click to restore it. No configuration files. No learning curve.

  • Captures window positions every 30 seconds and immediately before sleep
  • Restores windows after wake, display changes, or app launches
  • Matches displays by resolution — survives the display ID shuffle macOS does after every sleep
  • Saves named layouts — switch between "Work", "Coding", "Presentation" in one click
  • Runs silently in the menu bar, no Dock icon, no interruptions

What Gets Installed

  • /Applications/WindowLock.app
  • ~/Library/LaunchAgents/com.way2do.windowlock.plist (auto-start on login)

Data Storage

  • Window state and saved layouts: ~/Library/Application Support/WindowLock/
  • Logs: /tmp/windowlock-stderr.log

Permissions

Accessibility permission is required to move and resize windows. You'll be guided through this after installation.

Screen Recording is optional and improves window title detection.
